## **Crate**
A package of Rust code.  
Binary crates → executables.  
Library crates → reusable code.

---

## **Cargo**
Rust’s build system and package manager.  
Used to create, build, run, and test projects.

Commands introduced:
- `cargo new`
- `cargo run`
- `cargo build`
- `cargo check`

---

## **`main` function**
The entry point of every Rust executable program:
```rust
fn main() { }

---

## **macro**
Code that expands before compilation.  
Macros always end with `!`.

---

## **println!**
A macro used to print text to the console.

---

## **std**
The Rust standard library, containing core modules like I/O and comparison utilities.

---

## **immutable variable**
The default in Rust — variables cannot change unless explicitly marked as mutable.

---

## **shadowing**
Creating a new variable with the same name as an existing one.

---

## **constant (const)**
An unchanging value declared with `const`, requiring a type annotation.

---

## **mutable variable**
Declared with `mut`, allowing the value to be changed.
